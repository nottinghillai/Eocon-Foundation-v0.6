# Iterators

_Source page: 666_



We can iterate over collections in contexts such as for loops and array comprehen-


sions. To demonstrate various iterators, we will use the


collect


function, which


returns an array of all items generated by an iterator:


julia>


"feed"


"sing"


"ignore"


];


julia>


collect(enumerate(X))


# return the count and the element

(1, "feed")


(2, "sing")


(3, "ignore")
