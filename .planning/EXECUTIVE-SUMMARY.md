## The Architecture

### Five-Folder Foundation

Every Foundation project has this structure:

1. **Constitution/**: LaTeX documents (the source of truth)
2. **Plans/**: Auto-generated Markdown task files
3. **Guides/**: LLM self-documentation of implementation
4. **Docs/**: External documentation (APIs, libraries, auto-scraped)
5. **Design/**: Shared code, repositories, reusable components

### Key Components

- **Main Orchestrator**: Analyzes requests, determines constellation needs
- **The Planner**: "Compiles" Constitutions into executable Plans (dependency graphs, parallel/sequential tasks)
- **Distributed Network**: Computers worldwide contribute compute, earn tokens
- **Testing Backbone**: Automated test generation and validation (the foundation of autonomous execution)
- **Entanglement Layer**: APIs, data contracts, integration infrastructure
