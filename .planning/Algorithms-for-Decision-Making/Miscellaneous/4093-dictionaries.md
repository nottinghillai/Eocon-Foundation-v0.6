# Dictionaries

_Source page: 659_



dictionary


is a collection of key-value pairs. Key-value pairs are indicated with a


double arrow operator


=>


. We can index into a dictionary using square brackets,


just as with arrays and tuples:


julia>


Dict


();


# empty dictionary


julia>


x[


# associate key 3 with value 4


julia>


Dict


=>


=>
