# Vectors

_Source page: 652_



vector


is a one-dimensional array that stores a sequence of values. We can


construct a vector using square brackets, separating elements by commas:


julia>


[];


# empty vector


julia>


trues(


);


# Boolean vector containing three trues


julia>


ones(


);


# vector of three ones


julia>


zeros(


);


# vector of three zeros


julia>


rand(


);


# vector of three random numbers between 0 and 1


julia>


];


# vector of integers
